### DESCRIPCIÓN TÉCNICA DE LA OBRA

**SW BALOTO - LotoPredictor PRO** es un software de análisis estadístico y generación de predicciones para los juegos de lotería Baloto y MiLoto de Colombia.

#### Componentes del Software:

1. **Módulo Principal (app.py)**
   - Aplicación web desarrollada con Flask Framework
   - API RESTful con múltiples endpoints para análisis predictivo
   - Sistema de almacenamiento de resultados históricos
   - Servidor web integrado

2. **Módulo de Lógica Predictiva (logic.py)**
   - Algoritmos de cálculo de frecuencias
   - Sistema de selección ponderada por frecuencia
   - Análisis de números calientes y fríos (Hot/Cold)
   - Cálculo de brechas (gaps) entre apariciones
   - Análisis de tendencias temporales
   - Análisis de frecuencia de pares
   - Distribución de sumas
   - Análisis posicional
   - Análisis específico de Super Balota

3. **Módulo de Obtención de Datos (scraper.py)**
   - Sistema de web scraping para obtención de resultados
   - Integración con fuentes de datos de Baloto y MiLoto

4. **Interfaz de Usuario (templates/index.html, static/)**
   - Interfaz web moderna y responsiva
   - Sistema de pestañas para navegación
   - Visualización de predicciones con animaciones
   - Gráficos interactivos con Chart.js
   - Diseño visual profesional con CSS personalizado

5. **Sistema de Distribución**
   - Ejecutable independiente (SW_BALOTO.exe)
   - Configuración de empaquetado con PyInstaller
   - Documentación de usuario

#### Tecnologías Utilizadas:
- Python 3.x
- Flask (Framework Web)
- BeautifulSoup4 (Web Scraping)
- Chart.js (Visualización de Datos)
- HTML5, CSS3, JavaScript
- PyInstaller (Empaquetado)
